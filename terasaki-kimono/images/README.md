# 画像フォルダ

このフォルダには、ウェブサイトで使用する画像ファイルを配置してください。

## 必要な画像ファイル

以下の画像ファイルを準備してください：

1. **hero.jpg** (1920x1080推奨)
   - ヒーローセクションの背景画像
   - 着物姿で楽しそうにしている女性のイメージ

2. **teacher.jpg** (600x800推奨)
   - プロフィールセクションの講師写真
   - 寺﨑今日子先生の写真

3. **background.jpg** (1920x1080推奨)
   - メッセージセクションの背景画像
   - 和の雰囲気の背景画像

4. **signature.png** (100x100推奨、透過PNG推奨)
   - メッセージセクションのサイン画像
   - 透明背景のPNG形式が推奨されます

## 注意事項

- 画像が存在しない場合、フォールバックとして外部の画像URLが使用されます
- 画像のファイル名は大文字・小文字を区別します
- 推奨サイズは目安です。実際の画像は適切にリサイズされます
